Collection of small test-cases and tools

  1. Notification Generator

    A tool for dynamically creating notifications in accordance with the Notifications API living standard.

  2. Push Message Generator

    A tool for dynamically creating, displaying and sending push messages via the Push API, RFC8030 and RFC8292.

  3. Push Encryption Verifier

    A tool for verifying Web Push encryption implementations using RFC8291 and RFC8188.
